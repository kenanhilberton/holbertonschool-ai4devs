# Microservices Architecture

- **Auth Service**: Manages user credentials, authentication, and security tokens.
- **Passenger Service**: Handles passenger profiles, trip history, and preferences.
- **Driver Service**: Manages driver registration, vehicle documents, and verification status.
- **Ride Matching Service**: Core algorithm for connecting drivers with passengers in real-time.
- **Payment Service**: Processes financial transactions, wallets, and cost-sharing.
- **Notification Service**: Sends real-time push notifications, SMS, and email alerts.
- **Sustainability Service**: Tracks CO2 savings and environmental impact metrics.
- **Analytics Service**: Collects system data for business reporting and performance monitoring.
